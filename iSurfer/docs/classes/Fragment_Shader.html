Fragment_Shader Class

Defined in: /Users/Dani/Documents/Workspace/Proyecto Final/isurfer/trunk/iSurfer/resources/shaders/fs1.glsl:7
Module: OpenGL

Last modified on January 13 2013 by dazar

This is the fragment Shader. It is in charge of all the calculations to render an algebraic surface. This Shader is not complete, it must be filled with the degree, equation and partial derivatives of the surface. It use an adaptation of a Ray Tracer to generate the image from the surface roots. It is based on the Surfer made by Christian Stussak and adapted by Daniel Azar and Cristian Prieto for the iPhone gpu. Some bug fix made also by Christian Stussak.

Polynomial

Defined in /Users/Dani/Documents/Workspace/Proyecto Final/isurfer/trunk/iSurfer/resources/shaders/fs1.glsl:209

Usage: s = firstrootDescartes(p,epsilon, tmpCoeffs);

This function shifts and strech the polynomial. The idea is to focus the roots in the interval (0;1) for precision. Shift is like using p(x-shift) instead of p(x). Strech is like using p(x/scale) instead of p(x).

Parameters:

  • p Polynomial

    polynomial to find the roots.

  • epsilon Highp float

    this should be related to the interval length.

  • tmpCoeffs Polynomial

    polynomial output / input, is the poly streched and shifted.

Returns:

Polynomial: If the return value is >= 0 then it is the first root, else it should be discarded.
